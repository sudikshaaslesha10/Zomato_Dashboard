## 🧩 Problem Statement  
Businesses often struggle with fragmented data across multiple sources, making it difficult to track KPIs, identify trends, and make timely decisions. Manual reporting further slows down insights and introduces inconsistencies.

**Goal:**  
Design a centralized, interactive dashboard to monitor performance, improve visibility, and enable data-driven decision-making.

---

## 🎯 Objectives  
- Consolidate and transform raw data into a structured format  
- Build an interactive dashboard for KPI tracking  
- Enable stakeholders to explore trends dynamically  
- Automate reporting workflows to reduce manual effort  

---

## 🛠️ Tools & Technologies  
- **Power BI** – Dashboard development & visualization  
- **Power Query** – ETL (data cleaning & transformation)  
- **DAX** – Measures and calculated metrics  

---

## 🔄 Data Pipeline (ETL Process)    
1. Cleaned and transformed datasets using Power Query  
2. Built relationships and data models for efficient querying  
3. Created calculated columns and measures using DAX  
4. Automated dashboard refresh for consistent reporting  

---

## 📊 Dashboard Features  

### 🔹 KPI Tracking  
- Monitors key business metrics in a single view  
- Highlights performance trends and anomalies  

### 🔹 Interactive Analysis  
- Slicers and filters for dynamic exploration  
- Drill-down capability for detailed insights  

### 🔹 Bookmark Navigation  
- Custom navigation using bookmarks  
- Enables storytelling and structured data exploration  

---

## 📈 Key Insights  
- Identified performance trends across time and segments  
- Highlighted inefficiencies and areas of improvement  
- Enabled faster and more accurate decision-making  

---

## 🚀 Business Impact  
- Reduced manual reporting effort through automation  
- Improved data visibility for stakeholders  
- Enhanced decision-making speed with real-time insights

## 🧠 Key Learnings  
- End-to-end BI workflow: ETL → Modeling → Visualization  
- Importance of clean and validated data for accurate insights  
- Designing dashboards for both technical and non-technical users  
- Leveraging DAX for advanced analytics
